# Project Summary

## Project Name
MCP Skills To-Do App

## Project Type
Small React proof-of-concept app

## Project Goal
Build a simple to-do app that demonstrates a realistic software workflow using MCPs and skills.

## Core Features
- Add a task
- Mark a task as completed
- Delete a task
- Filter tasks by All / Active / Completed

## Out of Scope
- Login
- User accounts
- Backend
- Database
- Cloud storage
- Authentication
- Advanced styling
- Extra features not required for the assignment

## Project Description
I am building a small React to-do app for this assignment. The app allows users to add tasks, mark tasks as completed, delete tasks, and filter tasks by All, Active, and Completed.
